Lumi

Ein kleiner KI-Assistent, der im System-Tray wartet, bei Bedarf als dezente Pille erscheint und Diktat sowie sichere Textvorschlaege direkt in deinen Alltag bringt.

Das Light-Prinzip

So wenig UI wie moeglich, so viel KI wie noetig: keine grossen Fenster, kein Fokusdiebstahl, Mikrofon nur aktiv, wenn du Lumi wirklich aufrufst.

In wenigen Minuten startklar

Lumi ist eine native Windows-App. Nach dem Start erscheint sie unten rechts im System-Tray. Beim ersten Start oeffnen sich die Einstellungen, wenn noch keine API-Keys hinterlegt sind.

Lumi herunterladen

Speichere die EXE und starte sie per Doppelklick. Windows kann bei neuen lokalen Builds eine Sicherheitsabfrage anzeigen.

Keys eintragen

Hinterlege einen Groq-Key fuer Spracherkennung und einen OpenRouter-Key fuer KI-gestuetzte Textvorschlaege.

Mit Win+J sprechen

Halte Win + J, sprich dein Diktat oder deine Änderungsanweisung und lasse los. Lumi verarbeitet alles im Hintergrund.

Was du mit Lumi machst

Ein Hotkey, zwei Modi. Lumi merkt sich deine Auswahl; der Wechsel klappt in der Pille oder per Doppeltipp auf Win + J.

Vorschlag

Markierten Text umschreiben

Markiere Text, halte Win + J und sage zum Beispiel: "Mach das freundlicher". Lumi zeigt zuerst eine Vorschau und ersetzt nur die geprüfte Auswahl.

Diktat

Direkt am Cursor diktieren

Setze den Cursor an die Zielstelle, halte Win + J und sprich. Lumi nutzt dein persönliches Wörterbuch und fuegt den Text direkt dort ein.

Die wichtigsten Tasten

Win + J kurz antippen

Die kompakte Pille ein- oder ausblenden.

Win + J halten

Push-to-Talk: Lumi nimmt nur auf, solange du den Hotkey haeltst.

Win + J doppelt antippen

Zwischen Vorschlag und Diktat wechseln.

Esc

Overlay schliessen oder eine laufende Aktion abbrechen.

API-Keys bekommen

Lumi speichert Keys lokal unter %APPDATA%\Lumi\ und verschluesselt sie per Windows DPAPI. Teile Keys nie in Chats, Screenshots oder Git-Repositories.

Groq

Fuer Spracherkennung

  1. Oeffne console.groq.com und melde dich an.
  2. Gehe zu API Keys.
  3. Erstelle einen neuen Key und kopiere ihn direkt nach dem Erstellen.
  4. Oeffne Lumi, gehe in die Einstellungen und fuege ihn im Groq-Feld ein.
OpenRouter

Fuer Textvorschlaege

  1. Oeffne openrouter.ai und melde dich an.
  2. Gehe zu API Keys.
  3. Erstelle einen Key, optional mit Credit-Limit gegen unerwartete Kosten.
  4. Oeffne Lumi, gehe in die Einstellungen und fuege ihn im OpenRouter-Feld ein.

Wichtig zu Datenschutz und Kosten

Mikrofon nur bei Bedarf

Lumi nimmt nicht dauerhaft auf. Das Mikrofon ist nur aktiv, wenn du den Hotkey nutzt oder eine Aufnahme bewusst startest.

Clipboard wird wiederhergestellt

Bei Textbearbeitung sichert Lumi die Zwischenablage, nutzt sie kurz fuer Lesen oder Einfuegen und stellt sie danach wieder her.

Niedrige laufende Kosten

Groq uebernimmt die Spracherkennung; OpenRouter wird nur fuer Textvorschlaege gebraucht. Normales Diktat kommt ohne LLM-Aufruf aus.
